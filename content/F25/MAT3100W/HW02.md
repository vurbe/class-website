### [5 pts] Choice Exercise 8.
<p><i>Only attempt this exercise if you're feeling quite confident about how proofs by induction work.</i></p>
Here's a classic (incorrect) induction proof of the (obviously) false claim: "all horses are the same color."
The proof has a subtle error in it. Can you spot it?

<div style="border: 1px solid gray;">
<p><b>Claim: given any group of \(n\) horses, they all are the same color.</b></p>
<p>
  <i>Proof:</i>
  We proceed by induction, where the base case is clear: when \(n=1\), we have a single horse, and so all of the horses in that group are the same color.
</p>
<p>
  Assume (as an induction hypothesis), that any group of \(n\) horses is the same color for all \(n = 1, 2, \dots, k\).
</p>
<p>
  Suppose (as an induction hypothesis) that we have \(k+1\) horses, and we label them \(h_1, h_2, \dots, h_{k+1}\). By the induction hypothesis, we know that the \(k\) horses \(h_1, h_2, \dots, h_k\) are all the same color. Moreover, we know that the \(k\) horses labeled \(h_2, h_3, \dots, h_{k+1}\) are all the same color.
  Thus \(h_1\) is the same color as \(h_2\), which is the same color as \(h_{k+1}\), and so all \(k+1\) horses are the same color.
</p>
<p>
  Thus since whenever \(k\) horses are the same color, we can conclude that \(k+1\) horses are the same color, it must be the case that all horses are the same color. \(\square\)
</p>
</div>
